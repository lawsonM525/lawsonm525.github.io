Midterm 1 Components

Project Overview

In this midterm, I implemented three interactive components:

  • An Icon Bar component for navigation
  • An Accordion component for organized downloads
  • A Tabs component for content organization

You can learn more about each component in the tabs above, or check out their implementations here:

Icon Bar Component

The Icon Bar is a vertical navigation component that uses Font Awesome icons. It includes:

  • Vertical layout with fixed positioning
  • Nice smooth hover transitions
  • Icons from the font awesome library
  • Links to the main sections: Home, Photos, Movies, Downloads, and Print

Accordion Component

Accordion content is collapsible. Key features:

  • Dynamic section creation using a global variable called NRIMAGES
  • Smooth expand/collapse animations
  • Organized file downloads

Tabs Component

The Tabs system (which you're currently using) organizes content into separate views. Features:

  • Nice clean interface
  • Smooth transitions between tabs
  • Active tab highlighting
