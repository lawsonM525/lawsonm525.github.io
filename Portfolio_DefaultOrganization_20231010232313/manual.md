# Pink Themed Website User Manual

## Introduction

The Pink Themed Website is a modern and stylish website that showcases information about a person named Michelle Lawson and provides space for her portfolio. The website has a pink color scheme and features rounded corners on all elements. This user manual will guide you through the installation process and explain how to use the website.

## Installation

To use the Pink Themed Website, you need to install the required dependencies. Follow the steps below to install the necessary software:

1. Install Python: The website is built using Python programming language. If you don't have Python installed, download and install the latest version from the official Python website (https://www.python.org).

2. Install tkinter: The website uses the tkinter library for creating the graphical user interface. tkinter is included with Python, so you don't need to install it separately.

3. Download the website files: Download the website files from the provided source and save them in a directory on your computer.

## Usage

Once you have installed the required dependencies and downloaded the website files, you can start using the Pink Themed Website. Follow the steps below to run the website:

1. Open a terminal or command prompt.

2. Navigate to the directory where you saved the website files.

3. Run the following command to start the website:

   ```
   python main.py
   ```

4. The website will open in a new window. You can navigate through the website by clicking on the buttons and links.

5. To view Michelle Lawson's portfolio, click on the "View Portfolio" button. This will open a new window with detailed information about her portfolio.

6. To close the website, simply close the window.

## Customization

If you want to customize the website, you can modify the HTML, CSS, and JavaScript files. The main HTML file is `index.html`, and the portfolio HTML file is `portfolio.html`. The CSS styles are defined in `styles.css`, and the JavaScript code is in `index.js`.

You can change the color scheme, fonts, layout, and content of the website by editing these files. Make sure to save your changes and restart the website to see the updates.

## Conclusion

The Pink Themed Website is a stylish and modern website that showcases information about Michelle Lawson and her portfolio. Follow the installation instructions and use the website to explore Michelle's background and achievements. Enjoy using the Pink Themed Website!
